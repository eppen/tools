JSON 格式化

格式化、压缩或校验 JSON 字符串，解析失败时提示大致行号。

使用说明

将 API 响应或配置文件粘贴到左侧输入框，点击「格式化」以 2 空格缩进美化 JSON，或点击「压缩」移除空白字符。若 JSON 语法有误，工具会显示错误信息并尝试标注出错行号。所有处理均在本地浏览器完成，不会上传到任何服务器。

  • 支持对象、数组、字符串、数字、布尔与 null
  • 适合调试 REST API、配置文件与日志片段
