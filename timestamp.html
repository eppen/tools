Unix 时间戳转换

将 Unix 时间戳（秒或毫秒）转换为可读日期，或获取当前时间戳。

—
—
—

使用说明

Unix 时间戳表示自 1970-01-01 00:00:00 UTC 起的秒数或毫秒数。10 位数字通常为秒，13 位通常为毫秒。输入时间戳后点击「转换」，或使用「当前时间」一键填充。工具会同时显示您浏览器所在时区的本地时间、UTC 字符串与 ISO 8601 格式，便于日志分析与 API 调试。

  • 支持负时间戳（1970 年之前）
  • 按 Enter 键可快速转换
